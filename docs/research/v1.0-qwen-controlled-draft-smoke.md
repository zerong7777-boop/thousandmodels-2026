## Claim

v1.0 uses Qwen only as an optional controlled draft generator. Model output is schema-checked, public-copy checked, recorded as `AgentModelCall`, and replaced by deterministic fallback when unavailable or unsafe.

## Non-Claims

- No QwenPaw workflow is implemented.
- Qwen does not create or approve plan versions.
- Qwen does not publish public notices.
- Qwen does not mutate inventory or merchant runtime state.
- The demo still runs without `DASHSCOPE_API_KEY`.
