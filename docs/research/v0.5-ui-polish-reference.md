## Recommended Visual Direction

Use the same implementation direction across v0.5:

```text
owned Tailwind product components
+ shadcn/ui block structure
+ satnaing/shadcn-admin density reference
+ role-specific visual rhythm
```

Role-specific rhythm:

| Role | UI rhythm | Main references |
| --- | --- | --- |
| Organizer | Dense operations console with queues, metrics, timelines, approval panels | shadcn blocks, satnaing, Linear, Datadog, Tremor |
| Merchant | Mobile/tablet workbench with large direct actions and status feedback | Clover, Flowbite |
| Tourist/public | Mobile cultural route with event identity, route progress, stories, and live notices | Luma, Wanderlog, Roadtrippers |

## Do Not Copy

- Proprietary screenshots, brand colors, exact layouts, imagery, or commercial product wording.
- Full admin apps, routers, auth providers, chart stacks, maps, POS/payment flows, or ticketing features.
- Ant Design Pro shell or generic CRUD/resource admin structure.
