# v0.7 Vision-To-MVP Gap

Date: 2026-06-12
Status: draft for competition demo preparation

## Original Vision Summary

The original engineering overview describes Zhiyin Haojiang as a multi-agent collaborative operations platform for Macau old-district cultural-tourism events. It covers organizer planning, specialist Agent collaboration, merchant Edge Agent onboarding, tourist H5 publication, live recovery, review reports, and long-term knowledge accumulation.

## Current MVP Summary

The current system is a multilingual deterministic MVP. It has a FastAPI backend, Pydantic schemas, SQLite persistence, backend-backed demo auth, role-separated React/Vite frontend surfaces, public H5, plan versioning, incident detection, recovery proposal approval, public notice publication, and metric-backed review. It does not require `DASHSCOPE_API_KEY`.

## How To Explain The Gap

The demo should say that the current MVP proves the critical closed loop first. Full multi-agent Qwen/QwenPaw capability is a planned advisory layer after the deterministic workflow is stable. This is a deliberate engineering choice: judges can reproduce the demo locally without model availability, while the architecture still leaves clear insertion points for model-assisted planning and wording.

## Strongest Current Evidence

1. Role-separated product surfaces prove the system is not a single demo page.
2. Merchant status updates can trigger organizer-facing incidents.
3. Recovery approval creates a new plan version and public notice.
4. Public H5 hides backend concepts from visitors.
5. Review report connects metrics, incidents, and recommendations.
6. Tests and screenshots verify the deterministic path.

## Recommended Next Technical Phase After v0.7

After the narrative package is reviewed, the next technical phase should be a narrow Qwen/QwenPaw advisory spike. The first model-backed feature should draft recovery explanations, public notices, or review prose, while deterministic rules remain the source of truth for plan versioning, approval, inventory, and publication state.
