## Scenario

The Playwright smoke covers the v1.2 judge-facing loop:

1. Organizer opens the event workspace.
2. Organizer drafts and publishes the Event Page.
3. Organizer generates merchant edge packages.
4. Merchant reviews the per-shop package, touchpoints, and coupon rule.
5. Merchant reports sold-out runtime state.
6. Organizer opens the exception center and inspects an operation suggestion.
7. Public H5 opens as a visitor Event Page.
8. Public H5 simulates scan, coupon claim, and redemption.
9. Review center shows touchpoint summary, merchant outcomes, and extension tasks.
